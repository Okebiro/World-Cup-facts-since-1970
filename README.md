# World-Cup-facts-since-1970
This is an analysis of data from Kaggle that covers events during the World Cup from 1970 to 2022
## Facts to be discovered
1. How many World Cups have been played in that period?
2. How many teams have won since?
3. Top 10 matches with the most goals.
4. Top 5 finals with the most goals.
5. How many finals ended in penalty shootouts?
6. How many finals ended during normal time?
## Key Insights
1. There have been 14 World Cups.
2. Argentina has the most wins in the finals.
3. Three finals have ended in penalty shootouts.
